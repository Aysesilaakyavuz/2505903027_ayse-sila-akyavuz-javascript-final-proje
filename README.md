 CherubTV - Dizi & Film Keşif Platformu
CherubTV, TVMaze API üzerinden veri çekerek çalışan, modern ve interaktif bir dizi/film listeleme uygulamasıdır. Kullanıcıların yapımları keşfetmesi, filtrelemesi ve favorilerini saklaması için tasarlanmıştır.
 Temel Özellikler
• Dinamik Veri Çekme: JavaScript fetch API kullanılarak gerçek zamanlı veri yönetimi sağlanmıştır.
• Arama & Filtreleme: Arama çubuğu ve kategori butonları ile anlık sonuç listeleme.
• Detay Modalı: Sayfa değiştirmenize gerek kalmadan, kartlara tıklayarak yapımların özet bilgilerine ulaşabilirsiniz.
• Tam Responsive: Mobil ve masaüstü cihazlarla %100 uyumlu arayüz.
 Bonus (Ekstra) Özellikler
• Tema Yönetimi: Dark/Light mod geçişi ve kullanıcı tercihinin saklanması.
• Favori Listesi: localStorage entegrasyonu ile favori dizilerinizi tarayıcı kapansa bile kaybetmezsiniz.
• Sıralama: Yapımları alfabetik olarak (A-Z / Z-A) sıralayabilme imkanı.
• Sayfalama: "Daha Fazla Yükle" sistemi ile kartların performanslı şekilde listelenmesi.
 Dosyalar
• index.html: Sayfa yapısı ve semantik bölümler.
• style.css: Modern tasarım ve tema ayarları.
• script.js: API bağlantısı, DOM yönetimi ve bonus fonksiyonlar
